HOW TO USE

  1. Load jQuery and include all the fat.scrolls plugin files

    To use fat.scrolls plugin, it is important to include both the plugin and jQuery 1.7+ scripts.

    <!-- Important fat.scrolls stylesheet -->
<link rel="stylesheet" href="fat-scrolls/fat.scrolls.css">

<!-- Load the jQuery plugin -->
<script src="jquery-1.9.1.min.js">

<!-- Include the fat.scrolls script file -->
<script src="fat-scrolls/fat-scrolls.js">

  2. Set up your HTML

    For the HTML, all you need to do is give the div which includes the ul list a class fat-scroll and the ul list a class fat-scroll-list.

  3. Call the plugin

    Now, just call the function on any item you want with your preferred options in a JSON object.

    $('#fat-scroll-demo').fatScroll({
    style    :    'shrink'
    width    :    '300px'
    height   :    '450px'
});

Customizing Options

Option Default Type Description
style shrink string This aspect defines the animation style of the list elements when one scrolls up or down.
width 270px integer/string This defines the width of the outer container. The list is by default 100% of the width of this container. (It can be an integer like 300 or a string like '300px' which would mean the same thing. You can also set the width in percentage like '50%')
height 500px integer/string This defines the height of the list and not the outer container. The outer container has a height 'auto' by default. (It can be an integer like 500 or a string like '500px' which would mean the same thing. You can also set the height in percentage like '50%')
