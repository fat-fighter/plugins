A stylish jQuery plugin that allows you to easily add a courousel on your webpage Download Check it out on Github

EXAMPLE


HOW TO USE

  1. Load jQuery and include all the fat.carousel plugin files

    To use fat.carousel plugin, it is important to include both the plugin and jQuery 1.7+ scripts.

    <!-- Important fat.carousel stylesheet -->
<link rel="stylesheet" href="fat-carousel/fat.carousel.css">

<!-- Load the jQuery plugin -->
<script src="jquery-3.1.1.min.js">

<!-- Include the fat.carousels script file -->
<script src="fat-carousel/fat-carousel.js">

  2. Set up your HTML

    For the HTML, all you need to do is give the div which includes the main containers. You can also include any images you want directly in the html, other than specifying them while calling the plugin.

    <div class="fat-carousel" id="slideshow">
    <div class="fat-carousel-container-outer">
        <div class="fat-carousel-container">
        </div>
    </div>
</div>

  3. Call the plugin

    Now, just call the function on any item you want with your preferred options in a JSON object.

    $('#slideshow').fatCarousel({
    'imageArray'      :    ["images/1.jpg", "images/2.jpg", "images/3.jpg", "images/4.jpg", "images/5.jpg"],
    'itemsOnScreen'   :    [[992, 2], [500, 1]],
    'startSlide'      :    1,
    'slideTime'       :    400
});

Customizing Options

Option Default Type Description
style shrink string This aspect defines the animation style of the list elements when one scrolls up or down.
width 270px integer/string This defines the width of the outer container. The list is by default 100% of the width of this container. (It can be an integer like 300 or a string like '300px' which would mean the same thing. You can also set the width in percentage like '50%')
height 500px integer/string This defines the height of the list and not the outer container. The outer container has a height 'auto' by default. (It can be an integer like 500 or a string like '500px' which would mean the same thing. You can also set the height in percentage like '50%')
